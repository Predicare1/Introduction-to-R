# Introduction-to-R


This repository contains the files I used to create videos as part of a foundational biostatistics course at the University of Calgary. I was the lead teaching assistant (TA) for the R tutorials, so I have prepared a three-part introductory video series that will guide you through the basics of R and RStudio, helping you prepare to use R for data analysis.

The three-part videos are available on my YouTube channel below:

* Part 1 (Video): Getting Started with R: https://youtu.be/p1n82tlnYuw

* Part 2 (Video): Data Structures in R: https://youtu.be/vK3lzPS_NmU

* Part 3 (Video): R Packages and Importing Data in R: https://youtu.be/Ph7e24jybc4
